# 🐱 三体系整合命盘排盘工具

八字 + 紫微斗数 + 七政四余 三体系合一排盘工具。

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
# 网页版
streamlit run src/web_app.py

# 命令行
python3 src/cli.py -n "Gigi Wu" -d "1990-10-23" -t "07:00"
```

## 真太阳时

八字排盘需要精确的出生时间。中国横跨 60+ 经度，不同城市的太阳时差异很大（乌鲁木齐比北京慢 2 小时+）。

### 工作原理

```
真太阳时 = 北京时间 + (当地经度 - 120) × 4分钟
```

北京时间对应东经 120°，每偏 1° 修正 4 分钟。

### CLI 参数

```bash
# 方式1: 指定城市名（推荐）
python3 src/cli.py -n "某人" -d "1990-10-23" -t "07:00" -c "乌鲁木齐"

# 方式2: 直接指定经度
python3 src/cli.py -n "某人" -d "1990-10-23" -t "07:00" --lon 87.6

# 禁用真太阳时修正（使用原始北京时间）
python3 src/cli.py -n "某人" -d "1990-10-23" -t "07:00" --no-solar-time
```

### 网页版

在表单中填写"出生城市"，系统会自动查经度并计算真太阳时。留空则使用手动输入的经纬度。

### 测试案例

```
上海 (121.5°E) 7:00 北京时间 → 真太阳时 7:06（辰时）
乌鲁木齐 (87.6°E) 7:00 北京时间 → 真太阳时 4:54（丑时，不是辰时！）
```

### 其他参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--lat` | 纬度 | 31.3 |

## 晚子时规则

八字时辰的分界有两种主流规则：

| 规则 | 23:00-23:59 | 00:00-00:59 |
|------|-------------|-------------|
| **晚子时规则（默认）** | 属下一天的子时 | 属当天的丑时 |
| **标准/midnight规则** | 属当天的子时 | 属当天的丑时 |

本工具**默认启用晚子时规则**，即 23:00-23:59 出生时：
- 日柱算作**下一天**
- 时柱为**子时**

```bash
# 晚子时规则（默认）
python3 src/cli.py -n "某人" -d "1990-10-23" -t "23:30"
# → 日柱属下一天，子时

# 禁用晚子时，使用标准规则
python3 src/cli.py -n "某人" -d "1990-10-23" -t "23:30" --no-late-zichen
# → 日柱属当天，子时
```

**测试案例：**
```
1990-10-23 23:30（晚子时）：
  日柱: 壬戌（比标准规则多一天）
  时柱: 子时

1990-10-23 00:30（晚子时）：
  日柱: 辛酉（与标准规则相同）
  时柱: 丑时（sxtwl已正确，无需修正）
```
